# Flutter Student Registration Form

A simple Flutter application that demonstrates how to build a registration form with input validation using `Form` and `TextFormField`.

##  Features

* Student registration form UI
* Input validation for all fields
* Email format validation
* Phone number length check
* Password strength validation
* Success message using SnackBar

##  Tech Stack

* Flutter
* Dart

### Prerequisites

* Flutter SDK installed
* Android Studio / VS Code
* Emulator or physical device

## How It Works

* The app uses a `Form` widget with a `GlobalKey<FormState>`.
* Each field uses `TextFormField` with validation logic.
* Validators ensure correct user input before submission.
* On successful validation, a `SnackBar` confirms registration.
